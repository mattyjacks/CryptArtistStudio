Media Mogul

The flagship program. A professional-grade video editor, image editor, and AI-powered media production studio designed to compete with DaVinci Resolve - but free and open source.

✓ Multi-Track Timeline - Drag-and-drop clips with snap-to-grid, trimming, splitting, and ripple editing
✓ Node-Based Compositing - DAG editor for chaining visual effects and color corrections
✓ AI Studio Mode - Generate complete videos from a text prompt with auto-selected footage, voiceover, captions, and music
✓ Pexels Integration - Search and import stock photos and videos directly into your project
✓ GiveGigs Media Bucket - Royalty-free music and sound effects from the GiveGigs.com Supabase bucket
✓ Image Generation - Create images with DALL-E via the OpenAI API
✓ Podcast Studio - Simplified two-track audio editor with AI script generation and TTS voiceover
✓ FFmpeg-Powered Export - MP4, WebM, GIF, PNG sequence with configurable resolution, bitrate, and codec

👩‍💻 VCW

VibeCodeWorker

A full vibe-coding IDE built right into the suite. Modeled after VS Code and Windsurf, powered by Monaco Editor, and driven by your own AI API keys. Write code, get AI help, test, audit, and ship - all without leaving the app.

✓ Monaco Editor - VS Code's editor engine with syntax highlighting for 20+ languages, minimap, and bracket colorization
✓ File Explorer - Tree view of project directories with full filesystem access via Tauri
✓ AI Chat Panel - Context-aware coding assistant supporting OpenAI, Anthropic, Google, and custom endpoints
✓ Built-in Terminal - Shell access with command execution directly inside the IDE
✓ AI-Powered Testing - Automatic bug detection, edge case analysis, and coverage gap identification
✓ Web Audit - Lighthouse-style website analysis scoring Performance, Accessibility, SEO, and Best Practices
✓ Problem Scanner - Auto-detects console.log leaks, TODO comments, long lines, debugger statements, and empty catch blocks
✓ Cross-File Search - Regex-capable search and replace across entire projects

DemoRecorder

A screen recording and live streaming tool built for software demos, gaming clips, and content creation. Stream to Twitch, YouTube, and Google Meet with professional quality.

✓ Screen Capture - Record entire screen or specific windows at up to 4K resolution and 120 FPS
✓ Live Streaming - Stream to Twitch, YouTube Live, and Google Meet via RTMP
✓ Recording Controls - Start, Pause, Stop with live timer and resolution/FPS selectors
✓ Input Logger Plugin - Optional keystroke and mouse event logging for AI training data (with clear warnings)
✓ Publishing Links - Quick integration with GiveGigs.com and SiteFari.com for sharing
✓ Resolution Presets - 720p, 1080p, 1440p, 4K, or custom resolutions

ValleyNet

An autonomous AI agent inspired by OpenClaw. ValleyNet can browse the web, execute tasks, manage your integrations, and remember context across sessions. It's your personal AI assistant that lives right inside CryptArtist Studio.

✓ Natural Language Tasks - "Research competitors", "Send this to Discord", "Book a meeting for Thursday"
✓ Skills Marketplace - Install and manage agent skills via SKILL.md manifests
✓ Browser Automation - Embedded webview that ValleyNet controls to browse, fill forms, and scrape data
✓ Service Integrations - Discord, Slack, Telegram, WhatsApp, Gmail, Google Calendar, and webhooks
✓ Persistent Memory - Task history and context that carries across sessions

GameStudio

The newest addition to the suite. GameStudio combines Media Mogul, VibeCodeWorker, and Godot Engine into a unified game development environment. Describe a game concept and watch AI generate the code, scenes, and assets.

✓ Three-Panel Layout - Asset editor, code editor, and Godot Engine side by side
✓ Godot Integration - Auto-detect or download Godot, manage projects from within the studio
✓ AI Game Generation - Describe a game in natural language, get GDScript, scenes, and asset suggestions
✓ Project Templates - 2D platformers, top-down RPGs, puzzle games, and more
✓ Asset Pipeline - Generate sprites, textures, and UI with AI, then import directly into Godot
✓ Layout Modes - Split, Code Focus, Media Focus, and Godot Focus views
