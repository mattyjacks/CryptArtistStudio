Workspace Management

Multi-file workspaces, groups, and resource sharing

Multi-File Workspaces

Overview

CryptArtist Studio supports opening multiple .CryptArt files simultaneously through a workspace management system. This enables multi-project workflows, resource sharing between projects, and side-by-side editing.

Opening Multiple Files

  • Use the "Open .CryptArt Files..." button in the Suite Launcher to open a multi-file dialog
  • Each file creates a separate workspace
  • Maximum 69 workspaces can be open simultaneously
  • Each workspace maintains its own state and data

Workspace Bar

A global tab bar appears above the main content area, showing all open workspaces with their names and program icons. You can:

  • Click a tab to switch between workspaces
  • Right-click a tab to access context menu options
  • Close individual workspaces with the X button
  • Drag tabs to reorder them

Workspace Limit

The maximum number of concurrent workspaces is 69 (a nice number 😎). This allows for large multi-project workflows while maintaining system stability.

Workspace Groups and Resource Sharing

Overview

Workspaces can be combined into groups to enable resource sharing between projects. This allows you to:

  • Share media pools across projects
  • Share AI context and conversation history
  • Share settings and preferences
  • Organize related projects together

Creating Workspace Groups

To create a workspace group:

  1. Right-click on a workspace tab
  2. Select "Combine with..." from the context menu
  3. Choose another workspace to combine with
  4. The workspaces are now grouped and can share resources

Resource Sharing Types

Resource Type Description Use Case
Media Pool Shared images, videos, audio files Use the same assets across multiple projects
AI Context Shared conversation history and prompts Maintain AI context across related projects
Settings Shared preferences and configuration Apply consistent settings to grouped projects
Files Shared file references Reference files across projects
Chat History Shared AI chat conversations Maintain conversation context

Workspace Context Menu

Right-click on a workspace tab to access:

  • Combine with... - Create a workspace group
  • Share Media - Enable media pool sharing
  • Share AI Context - Enable AI context sharing
  • Share Settings - Enable settings sharing
  • Duplicate - Create a copy of the workspace
  • Close - Close the workspace

Workspace Features

Workspace Persistence

Your workspace configuration is automatically saved:

  • Open workspaces are restored on app restart
  • Workspace order and grouping are preserved
  • Active workspace is remembered

Program-Specific Workspaces

Different programs have different workspace features:

  • Media Mogul: Edit, Node Mode, Color, Audio, AI Studio, Podcast, Deliver
  • VibeCodeWorker: File Explorer, Editor, Bottom Panel, AI Chat
  • GameStudio: Split, Code Focus, Media Focus, Godot Focus

Switching Between Workspaces

  • Click workspace tabs in the tab bar
  • Use keyboard shortcuts (if configured)
  • Use Alt+Tab to cycle through all workspaces
Pro Tip: Use workspace groups to organize related projects. For example, group all video projects together to share a media pool, or group code projects to share AI context.
