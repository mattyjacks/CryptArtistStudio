Global Shortcuts

These shortcuts work everywhere in CryptArtist Studio:

Shortcut Action
Home Open Suite Launcher
Ctrl+1 Open Media Mogul
Ctrl+2 Open VibeCodeWorker
Ctrl+3 Open DemoRecorder
Ctrl+4 Open ValleyNet
Ctrl+5 Open GameStudio
Ctrl+6 Open CryptArt Commander
Ctrl+0 Open Settings
Ctrl+S Save current project
Ctrl+O Open project
Ctrl+N New project
Ctrl+W Close current workspace
Ctrl+Tab Switch to next workspace
Ctrl+Shift+Tab Switch to previous workspace
Ctrl+Shift+M Toggle Mummy (awaken/silence)
F11 Toggle fullscreen
F12 Open developer tools
